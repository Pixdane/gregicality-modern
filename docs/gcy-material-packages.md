# GCY Material Migration Packages

This index is generated from `docs/gcy-material-dedupe.csv`. Each migration package has its own table file under `docs/gcy-material-packages/`.

Package tables intentionally exclude entries whose `status` is `exists_in_gtceu`; GTCEu Modern-provided materials remain in the master dedupe CSV and id map only. The package is an implementation-routing category, not a full chemistry taxonomy.

Nuclear-related entries remain classified as `deferred/nuclear` even when their old source section was not `NUCLEAR MATERIALS`.
