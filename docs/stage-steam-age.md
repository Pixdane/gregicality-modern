# 蒸汽时代 (Steam Age) 对比报告

> 数据来源：GTCE-TJ-Fork（1.12.2）、gregicality-TJFork（1.12.2 附属）、GregTech-Modern 7.5.3（1.20.1）源码。
> 蒸汽时代指从开局到建立 LV 电力之前的阶段。

---

## 1. 开局流程

三者基本一致：燧石工具起步 -> 挖矿获取铜/锡/铁/煤 -> 制作青铜。

| 步骤 | GTCE | TJFork | GTM |
|------|------|--------|-----|
| 燧石工具 | 研钵/剑/镐/铲/斧/锄/刀 | 同 GTCE | 同 GTCE |
| 金属手工具 | 硬锤(4锭+棍)、扳手(5锭+锤)、锉刀(2板+棍)等 | **扳手改为 4 板**（昂贵扳手，默认开启） | 同 GTCE（扳手用 3 板+锤） |
| 制作板 | 2 锭 + 锤子 -> 1 板 | 同 GTCE | 同 GTCE |
| 制作棍/杆 | 锭 + 锉刀 -> 1 棍 | 同 GTCE | 同 GTCE |
| 锻铁获取 | 铁粒在熔炉烧成锻铁粒 | 同 GTCE | **铁锭在熔炉烧成锻铁锭**（更直接） |
| 青铜制作 | 3 铜 + 1 锡 = 4 青铜（合成台/合金炉/混合机） | 同 GTCE | 3 铜 + 1 锡 = 3 青铜（合成台）/ 4 青铜（混合机） |

TJFork 关键差异：
- 弯曲圆筒工具（6 锭制作）成为必需品，用于制作环、箔、曲面板、管道
- 圆环从"锭+锤子"改为"棍+小弯曲圆筒"，影响活塞、电机、传送带的前置
- 曲面板替代了部分配方中的普通板（桶、盔甲、转子、管道）
- 圆形单元替代了部分配方中的螺丝

> 决策：GT6 工艺系统暂缓，如果做也是做成一个可开关的独立 feature。

## 5. TJFork 独有蒸汽机器详述

以下三台机器为 TJFork 独有，GTM 没有直接等价（蒸汽矿机除外，GTM 有不同设计的版本）。

> 决策：三台机器均为**可以实现但低优先级**。

### 5.1 蒸汽泵（Steam Pump）

**源码**：`tjfork: machines/SteamPump.java`（264 行），注册 ID 2232

#### 工作机制

- 继承 `MetaTileEntity`（自定义蒸汽机器，非 SteamMetaTileEntity）
- 青铜外壳，仅 LP 版，没有 HP 版
- 蒸汽槽 16000 mB，仅接受蒸汽
- 输出流体槽 4000 mB
- 泵向下延伸管道，每 20 tick 向下延伸 1 格，最大 50 格
- 在泵头位置 BFS 搜索流体源方块，搜索半径 16 格
- 每个抽液周期（120 tick）抽取一个源方块
- 每次抽液消耗 1000 mB 蒸汽
- 需要蒸汽量 >= 4000 mB 才启动
- 无排气机制

#### 合成配方

```
"NLN"     N = 青铜中流体管(pipeMedium Bronze)
"NMN"     L = 青铜大流体管(pipeLarge Bronze)
"LRL"     M = 青铜外壳(BRONZE_HULL)
          R = 青铜转子(rotor Bronze)
```

#### 与 GTM 对比

GTM 没有蒸汽泵，泵从 LV 起（`PumpMachine`，LV~EV）。TJFork 蒸汽泵填补了蒸汽时代抽取流体的空白。

#### 迁移评估

- 值得移植：填补蒸汽时代流体抽取空白
- 实现复杂度中等：BFS 搜索流体源+管道延伸渲染，需在 GTM 的 machine 框架上重写
- 优先级：低

---

### 5.2 蒸汽混合机（Steam Mixer）

**源码**：`tjfork: machines/TileEntitySteamMixer.java`（81 行），注册 ID 2235

#### 工作机制

- 继承 `SteamMetaTileEntity`（标准蒸汽机器基类）
- 青铜外壳，仅 LP 版（`false` 参数），没有 HP 版
- 配方图：`MIXER_RECIPES`（与电力混合机共用）
- 物品槽：4 输入 + 1 输出
- 流体槽：蒸汽槽 16000 mB + 2 个输入流体槽（各 32000 mB）+ 1 个输出流体槽（32000 mB）
- 有排气机制（继承 SteamMetaTileEntity）
- LP 版不允许超频

#### 合成配方

```
"GRG"     G = 玻璃(Glass)
"GPG"     R = 青铜转子(rotor Bronze)
"PMP"     P = 青铜小流体管(pipeSmall Bronze)
          M = 青铜外壳(BRONZE_HULL)
```

#### 与 GTM 对比

GTM 没有蒸汽混合机，混合机从 LV 起（`registerSimpleMachines("mixer", ...)`，LV~UV）。TJFork 蒸汽混合机允许蒸汽时代进行混合配方。

#### 迁移评估

- 值得移植：让蒸汽时代可以做混合配方（青铜制作、化工前处理等）
- 实现简单（81 行），继承标准蒸汽机器基类，GTM 的 `SimpleSteamMachine` 可直接适配
- 优先级：低

---

### 5.3 蒸汽矿机（Steam Miner）

**源码**：`tjfork: machines/multi/miner/TileEntitySteamMiner.java`（285 行），注册 ID 4212

#### 工作机制

- 继承 `MetaTileEntity`，实现 `Miner` 接口（自定义实现，非 SteamMetaTileEntity）
- 青铜外壳，`STEAM_MINER_OVERLAY` 覆盖纹理
- 矿机类型 `Miner.Type.STEAM`：
  - 每方块耗时 8 tick（0.4 秒）
  - 采矿范围：当前 chunk（16×16）
  - 时运 0
  - 钻井液消耗 1 mB/tick
- 蒸汽消耗 16 mB/tick
- 需提供 DrillingFluid（钻井液），16 mB/tick
- 物品槽：4 输出槽（2×2）
- 排气机制：每次挖完一批方块后需排气，方向向上，伤害 6.0
- 不可旋转

#### 合成配方

```
"DSD"     D = 钻石(Diamond)
"SMS"     M = 青铜外壳(BRONZE_HULL)
"RSR"     S = 青铜中流体管(pipeMedium Bronze)
          R = 青铜转子(rotor Bronze)
```

#### 钻井液配方

三个版本完全一致：

| 版本 | 配方 |
|------|------|
| GTCE | 混合机：石头粉×1 + 润滑油 20 mB + 水 4980 mB -> 钻井液 5000 mB（64t, 16 EU/t） |
| TJFork | 同 GTCE（直接引用 GTCE 的 `Materials.DrillingFluid`） |
| GTM | 同配方；额外有蒸馏水版（用 DistilledWater 替代水，耗时 48t 更短） |

关键：钻井液需要润滑油，润滑油需要石油加工。TJFork 油藏系统在蒸汽时代无法开采（Fluid Drilling Plant 最低 MV），所以 TJFork 蒸汽矿机虽然有蒸汽时代的合成配方，但实际运转可能卡在钻井液供应上。

#### 与 GTM 蒸汽矿机对比

| 对比项 | TJFork 蒸汽矿机 | GTM 蒸汽矿机（LP） | GTM 蒸汽矿机（HP） |
|--------|---------------|-------------------|-------------------|
| 挖掘速度 | 8 tick/方块 | 320 tick/方块 | 240 tick/方块 |
| 工作范围 | 当前 chunk（16×16） | 4×4 | 6×6 |
| 蒸汽消耗 | 16 mB/tick | 16 mB/tick | 32 mB/tick |
| 钻井液 | 需要（1 mB/tick） | 不需要 | 不需要 |
| 压力等级 | 仅 LP | LP | HP |
| 合成 | 青铜外壳+青铜管+青铜转子+钻石 | 青铜砖外壳+青铜管+青铜小齿轮+钻石 | LP 版升级（钢板+锡合金管+钻石+钢小齿轮） |

TJFork 版本快 40 倍+挖整 chunk，但需要钻井液作为平衡。GTM 版本慢但无钻井液依赖，蒸汽时代可直接运转。

#### 迁移评估

- GTM 已有蒸汽矿机（LP/HP），功能等价但参数差异大
- TJFork 版本的设计取向不同（快速+大范围+钻井液 vs 慢速+小范围+无钻井液）
- 优先级：低（GTM 已有等价机器，TJFork 版本可作为可选配置或后续调整）

## 8. LV 电路

> 决策：LV 电路直接用 GTM 的体系。

### 8.1 电路板层级对比

| 层级 | GTCE | TJFork | GTM |
|------|------|--------|-----|
| 第 1 层 | 涂层板（木板+橡胶×2） | 涂层板（木板+橡胶×6） | 涂层板（木板+树脂×6） |
| 第 2 层 | 无 | 基础电路板（涂层板+铜线×8） | 基础电路板（涂层板+铜线×8） |

GTCE 直接用涂层板做电路，TJFork 和 GTM 都多一层"涂层板->基础电路板"。

### 8.2 组件对比

| 组件 | GTCE | TJFork | GTM |
|------|------|--------|-----|
| 真空管 | 纸+玻璃管+铜线 | 同 GTCE | 纸+玻璃管+铜线+**钢螺栓**；组装机产出 2~4 个 |
| 电阻 | 纸+铜线+煤粉 -> 3 个 | 橡胶+铜细线+铜单线+煤粉 -> **1 个** | 树脂+纸+铜线+煤粉 -> 2 个 |
| 二极管 | 锡细线+染料+镓 -> 4 个 | 铜细线+镓（无染料）-> 1 个 | 铜细线+砷化镓（仅组装机）-> 1~4 个 |
| 电容 | 组装机：塑料板+铝箔 -> 2 个 | 同 GTCE | 组装机：聚乙烯箔+铝箔 -> 8 个 |
| 晶体管 | GTCE 内部处理 | 同 GTCE | 组装机：硅板+锡细线 -> 8 个 |

### 8.3 LV 电路本体

| 版本 | 名称 | 配方 |
|------|------|------|
| GTCE | Basic Circuit | 电阻+**钢板**+真空管×2+**涂层板**+红合金线×3 |
| TJFork | Primitive Circuit | 电阻+**锻铁板**+真空管×2+**基础电路板**+红合金线×3 |
| GTM | Electronic Circuit LV | 电阻+**钢板**+真空管×2+**基础电路板**+红合金线×3 |

TJFork 用锻铁板（更早获得）但组件产出最少（电阻 1 个/次）。GTM 与 GTCE 布局相同，多了基础电路板层和钢螺栓需求。

### 8.4 化学蚀刻

TJFork 和 GTM 都引入了电路板化学蚀刻工艺（过硫酸钠 SodiumPersulfate / 氯化铁 IronChloride 或三氯化铁 Iron3Chloride），GTCE 没有。GTM 已有等价实现，无需额外迁移。

### 8.5 两条电路线

TJFork 和 GTM 都在 LV 段有两条电路线：

| 版本 | 第 1 条（蒸汽/早期可达） | 第 2 条（需组装机/化学蚀刻） |
|------|------------------------|---------------------------|
| TJFork | Primitive Circuit（手工，锻铁板） | Basic Electronic Circuit（电路组装机，CPU+Good Phenolic Board+8电阻+8电容） |
| GTM | T1 Electronic（手工/组装机） | T2 Integrated（组装机，ILC 芯片+电阻+二极管+锡螺栓） |
